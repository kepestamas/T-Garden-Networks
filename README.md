### Disclamer

Some adjustments were made for the networks in order to create a unified network format and resolve eventual problems, such as missing coordinates, duplicate nodes, etc. 
Some networks are generated, created in order to simulate possible topology network layouts on a variety of regions. All networks adhere to the same unified format as the sourced datasets. The generation process is described in detail in the T-Garden article.

For more information and individual network visualisations, visit TGarden.net

The plots are only for visualization purposes!
